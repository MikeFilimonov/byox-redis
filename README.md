# byox-redis
Redis is an in-memory data structure store often used as a database, cache, message broker and streaming engine. 
In this challenge I'm building my own Redis server that is capable of serving basic commands, reading RDB files and more.
Along the way getting feet wet with TCP servers, the Redis Protocol and more.

Theis Codecrafters redis project containing simplified Redis implementation.

See https://app.codecrafters.io/courses/redis/overview for details
